# typescript-node-jest-eslint-vscode

Project template for
- typescript
- node.js
- jest
- eslint
- vscode

## Usage

```
npm install

# run index.ts by ts-node
npm start

# test by jest
npm run test

# compile src/*.ts to dist/
npm run build
```


## vscode

Debug configurations
- Debug Jest Tests
    - for testing src/__test__/**/*.ts
- Debug Program
    - for debugging src/index.ts
